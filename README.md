# utils

## 简介

这是一个工具库，里面有很多好用的函数

- umd 模块

## 快速开始

### 引入

- pnpm 引入

> pnpm i @wormery/utils

- npm 引入

> npm i @wormery/utils

- yarn 引入

> yarn add @wormery/utils

## 为什么要用本项目?

里面有很多优秀的函数可以减少您掉的头发，让您的头皮更健壮，函数基本都调试完成如果出现了什么 bug 请联系我[github](https://github.com/mekeer)或给我提[issues](https://github.com/wormery/utils/issues),我会尽快修复的
